PREFERENCES
Variables conservées :
- user_login : (String) login cartographe connecté
- user_id : (int) ID cartographe connecté
- user_pass : (String) pass cartographe connecté
- user_keep : (boolean) conserver ou non les infos de connexion (login/pass) (défaut:false)
- tmp_batiment_id : (int) ID du bâtiment courant
- tmp_batiment_nom; (String) nom du bâtiment courant
- tmp_niveau_id = (int) ID nu niveau courant
- tmp_niveau_nom : (String) nom du cniveau courant
- tmp_repere_id : (int) ID du repère courant
